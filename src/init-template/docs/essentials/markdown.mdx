# Markdown Syntax

Docsy uses MDX, which lets you use standard Markdown with embedded components.

## Text Formatting

| Syntax | Output |
|--------|--------|
| `**bold**` | **bold** |
| `*italic*` | *italic* |
| `~~strikethrough~~` | ~~strikethrough~~ |
| `` `inline code` `` | `inline code` |

## Headings

```md
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
```

Headings automatically appear in the table of contents on the right side of the page.

## Links

```md
[External link](https://example.com)
[Internal link](/quickstart)
```

## Code Blocks

Use fenced code blocks with a language identifier for syntax highlighting:

````md
```javascript
function hello() {
  console.log('Hello, world!');
}
```
````

Renders as:

```javascript
function hello() {
  console.log('Hello, world!');
}
```
