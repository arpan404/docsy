# Components

Docsy includes a library of MDX components you can use directly in your documentation.

## Callouts

Use callouts to highlight important information:

<Note>
  This is a note callout for general information.
</Note>

<Warning>
  This is a warning callout for potential issues.
</Warning>

<Tip>
  This is a tip callout for helpful suggestions.
</Tip>

<Info>
  This is an info callout for additional context.
</Info>

## Accordion

<AccordionGroup>
  <Accordion title="What is Docsy?">
    Docsy is an open-source documentation framework built on Astro. It provides a complete solution for building documentation sites with MDX.
  </Accordion>
  <Accordion title="How do I deploy?">
    Run `npx docsy build` and deploy the output directory to any static hosting provider.
  </Accordion>
</AccordionGroup>

## Steps

<Steps>
  <Step title="Install">
    Run the install command for your package manager.
  </Step>
  <Step title="Configure">
    Edit `mint.json` with your site settings.
  </Step>
  <Step title="Deploy">
    Build and deploy to your hosting provider.
  </Step>
</Steps>
