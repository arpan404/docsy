# Welcome to Your Docs

This is your documentation site built with [Docsy](https://github.com), an open-source documentation framework powered by Astro.

<Tip>
  Edit the MDX files in the `docs/` directory to update your documentation.
</Tip>

## Features

<CardGroup cols={2}>
  <Card title="MDX Support" icon="file-text">
    Write documentation using MDX with built-in components for cards, tabs, callouts, and more.
  </Card>
  <Card title="Themes" icon="palette">
    Choose from 20 built-in themes or customize colors to match your brand.
  </Card>
  <Card title="Fast" icon="zap">
    Built on Astro for blazing-fast static sites with zero JavaScript by default.
  </Card>
  <Card title="Search" icon="search">
    Built-in full-text search so your users can find what they need.
  </Card>
</CardGroup>

## Next Steps

<Steps>
  <Step title="Edit your docs">
    Modify the MDX files in the `docs/` folder. Changes appear instantly in dev mode.
  </Step>
  <Step title="Configure navigation">
    Update `mint.json` to organize your sidebar navigation and site settings.
  </Step>
  <Step title="Deploy">
    Run `npx docsy build` and deploy the output to any static hosting provider.
  </Step>
</Steps>
